Тестовое задание. 
Необходимо создать приложение «доска объявлений». 
Приложение представляет из себя SPA и в его основе лежит парадигма CRUD. Страница должна отображать список объявления хранящихся в базе данных. В начале списка должны быть самые «свежие» объявления. По умолчанию на странице должно отображаться 10 объявлений, если не указано обратное. Объявление должно содержать такие данные как: заголовок, описание (вводить описание можно длинное, но отображать нужно только 255 символов. Если описание длиннее, после 255 символов отображать троеточие), картинка, имя автора, почта автора, контактный номер телефона автора. 
Один автор может иметь много объявлений, объявление может принадлежать только одному автору. Так же, на странице должна быть форма для добавления нового объявления. При добавлении объявления нужна проверка вводимых данных. При сохранении объявления оно должно появиться вверху списка. Если объявлений на странице больше, чем указанное ограничение - лишнее объявление нужно убрать. Для упрощения задачи допускается использование bootstrap+flex+JS+JQuery. Допускается использование любимого фреймворка (предпочтительно laravel, lumen или symfony), но это не обязательно! 
Тестовое задание принимается в формате гит репозитория на GitHub или Bitbucket. Репозиторий должен содержать docker или vagrant впил с конфигурациями.
